📊 Integraciones MJP - Análisis de Gráficas

Tipos de Widgets, Librerías y Integración de APIs

Resumen Ejecutivo

Este documento detalla la arquitectura técnica de 6 gráficas interactivas desarrolladas para visualizar la integración de datos de múltiples APIs externas. Cada gráfica combina datos de la API propia (MJP - Salarios) con APIs externas de terceros mediante una arquitectura de tres capas: Frontend (Widget) → Backend (Proxy API) → APIs Externas.

Conclusiones

Arquitectura de Integración: Todas las gráficas siguen el patrón de integración de tres capas (Frontend → Backend → APIs Externas) para evitar problemas de CORS y garantizar la seguridad de las credenciales.

Librerías Utilizadas: Chart.js para las 5 primeras gráficas (compatibilidad, facilidad de uso, documentación), Apache ECharts para la 6ª (mayor flexibilidad y capacidades avanzadas).

Widgets: Componentes Svelte que renderan elementos HTML nativos (<canvas> para Chart.js, <div> para ECharts).
